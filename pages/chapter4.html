What is Data Storage?

Data storage is the process of writing (saving) data to a storage device. By storeing the data, we make sure that we can always come back to it.

Main Storage

Main storage is a type of storage that are used to temporarly store data that will be processed by the cpu, such as the RAM

Backing Storage

Backing is a type of storage that are used to store data that will be needed later, typically slower to access than the main storage, but useally cheaper per byte. An exaple of backing storage would be a HDD
