Chapter 3: Storage Devices and Media

Optical storage media - continued

CD-RW and DVD-RW

The RW means these disks are a rewritable media and can be written over several times. Unlike CD-R/DVD-R, they don't become ROMS. THe recording layer uses a special phrase-changing metal alloy (often GeSbTE [Germanium-Antimony-Terbium alloy]; a number of different methods are used to produce these alloys). The alloy can switch between crysstalline phase and amorphous phase (non-crystalline), thus changing inpus reflectivity to light depending on the laser beam power. Spots are produced that can be read by a laser and then interpreted as 1s and 0s. The system allows data to be written, erased and rewritten many times.

Uses

  1. Used to record television programmes (like a video recorder), which can be recorded over time and time again.
  2. Not as wasteful as R format as more files/data can be added to at a later stage (with CD-R/DVD-R it is only possible to do a write operation once if you have already finalised the disc).
  3. Used in CCTV systems.

Disadvabtages

  1. Can be relatively expensive.
  2. It is possible to accidentally overwrite data

DVD-RAM

DVD-RAM uses a very different technology to CDs and DVDs. They have the following features:

  • Instead of a single, spiral track, they use a number of concentric tracks
  • The use of concentric tracks allows simultaneous read and write operations to take place
  • They allow numerous read and write operations (up to 100 times) and have greate longevity (over 30 years), which makes them ideal for archiving.
  • DVD-RAMs can be written to and read from many times.

The recording layer is made from a similar phase-changing materal as used in RW technology. When writing, a laser heats the phase-changing alloy on the disk to about 500°C to 700°C, changing the reflective properties from shiny to dull (i.e. pits). If the disk needs to be erased, a laser heats the surface to about 200°C to return the disk to its original shiny state. A low-power laser is used to read the written marks on the surface. The shiny and dull marks ('pits') represent data to a computer where they are interpreted.

Uses

  1. In recording devices such as satelite receivers to allow simultanous recording and playback.
  2. Used in camcorders to store movies.

Advantages

  1. They have a long life (estimated 20 years minimum life).
  2. It is possible to do a rewrite operation over 100 000 times (compare this to the RW formate, which only allows about 1000 rewrites).
  3. Writing on the DVD-RAMs is very reliable - they have in-built verification software, so the accuracy of the data is ensured.
  4. Very fast access if the files are fairly small.
  5. No need to finalise the disk.
  6. Very large capacity (about 10 GB if double-sided format is used).
  7. They offer the ability to read data at the same time as data is being written.

Disadvantages

  1. Not as compactible as R or RW formatel many systems won't recognise the DVD-RAM formate.
  2. Relatively expensive (about four times the cost of a DVD-RW disk).
  3. They have been superseded by newter technologies such as solid state memories

Blu-ray discs

Blu-ray discs are another example of optical storage media. However, they are fundamentally different to DVDs in their construction and in the way they carry out read-write operations.

The main differences are:

  1. A blue laser, rather than a red laser, is used to carry out read and write operationsl the wavelength of blue light is only 405 nanometers (compared to 650 nanometers for red light).
  2. Using blue lase light means that the 'pits' and 'bumps' can be much smaller, consequently, Blu-ray can store up to file times more data than a normal DVD.
  3. Blu-ray uses a single 1.1mm-thick polycarbonate disk; normal DVDs use a sandwich of two 0.6mm thick disks.
  4. Using two sandwich layers can cause birefringence (light is refracted into two seprate separte beams causing reading errors); because Blu-ray uses only one layer, the disks don't suffer from birefingence.
  5. Blu-ray discs automatically come with a secure encryption system, which helps to prevent piracy and copyright infringment.

The table below summarises the main differences between CDs, DVDs and Blu-ray.

Disk type Laser colour Wavelength of laser light Disk construction Track pitch (distance between tracks)
CD red 780nm single 1.2mm polycarbonate layer 1.60μm
DVD red 650nm two 0.6mm polycarbonate layer 0.730μm
Blu-ray blue 405nm single 1.1mm polycarbonate layer 0.30μm

Uses

  1. Home video consoles.
  2. Storing and playing back movies (one high-definition movie of two hours duration uses up 25GB of memory).
  3. PCs can use this technology for data storage or backing up hard drives.
  4. Camcorders can use this media (in a cartidge form) to store movie footage.

Advantages

  1. Very large storage capacity, therfore ideal for storing high- definition movies.
  2. Very fast data transfer rate.
  3. The data access speed is also greater than with other optical media.
  4. Blu-ray discs automatically come with a secure encryption system, which helpes to prevent piracy and copyright infringement.

Disadvantages

  1. Relatively explensive.
  2. Encryption problems (which are used to stop piracy) when used to store video
  3. There are fewer movie titles on the Blu-ray format, which is reducing its impact on the home movie market.

All these optical storage media are used as backup system (for photos, music and multimedia files). This also means that CDs and DVDs can be used to transfer files between computers. Manufacturers often supply their software using CDs and DVDs. When the software is supplied in this way, the disk is useally in a read only format.

The most common use of DVD and Blu-ray is the supply of movies or games. The memory capacity of CDs isn't big enough to store move movies.

The future of optical media

In recent times both the CD and DVD are shwing signs of becomeing obsolete. many computer systems now come with USB connectores only and no DVD or CD drive. The main method of transferring files between devices has become flash memory. Many people now store all their music in the following ways:

  • On hard disk drive systems
  • In MP3 format on:
    • A computer or tablet
    • Their mobile or smartphone
    • A portable music player
  • Using the 'cloud' to store all their files so they can access their music from anywhere in the world.
  • By 'streaming' their music from the internet; provided the user had an internet connection they can access music through a laptop computer, mobile phone, tablet or any other receiving devices. (i.e. Spotify, Soundcloud, ect)

It is a similar story for movies, where streaming is becoming increasingly common. Many television sets are now set up as 'smart' televisions - this means it is now possible to simply stream movies or television programmes on demand without the need for any DVD or Blu-ray players. In effect, the television set has become the central computer with a link to the internet using wireless connection.

Floppy disks met the same fate in the early twenty-first century. How often do you see floppy disks? It is very likely that CDs and DVDs will meet the same fate and be replaced by one of the systems described above or someting emtirely new.
